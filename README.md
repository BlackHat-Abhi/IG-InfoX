# IG-InfoX

Get Instagram account information with this tool

*Things that must be observed !!!*

   - This tool does not display accurate information about private accounts

# Installation

`git clone https://github.com/BlackHat-Abhi/IG-InfoX.git`

`cd IG-InfoX`

`python3 IG-InfoX -h`
